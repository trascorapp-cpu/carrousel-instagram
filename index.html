Comment utiliser l'outil ?

  1. Importez une grande image contenant toutes les slides de votre carrousel, côte à côte.
  2. Vérifiez ou corrigez le nombre de slides détecté.
  3. Cliquez sur « Convertir » pour découper l'image.
  4. Vérifiez l'aperçu, puis téléchargez le fichier ZIP contenant vos slides prêtes pour Instagram (1080 × 1350 px).

Tout se passe dans votre navigateur : aucune image n'est envoyée sur un serveur.
